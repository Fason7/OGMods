# Grenade
[Forum post.](http://forums.wolfire.com/viewtopic.php?f=16&t=37419)  
## Howto use:  
	1. Go to Load -> Interactive Objects -> Grenade.  
	2. Place it somewhere in the scene.  
	3. Pick up the grenade and throw it at an enemy.
